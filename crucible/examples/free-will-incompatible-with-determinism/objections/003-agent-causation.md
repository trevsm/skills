# Objection: Agent Causation

## Objection (Steelmanned)

Agent-causal theories claim that agents, not merely prior events, can originate actions. If agent causation is coherent, then an agent can be the ultimate source of action and deserve responsibility even if event-causal determinism is false or incomplete.

## Pass Criteria

The reply MUST:

- Address why agent causation does not rescue compatibility with determinism.
- Avoid treating agent causation as obviously incoherent unless argued.

The reply MUST NOT:

- Drift into a broader thesis that no free will exists under any metaphysics.
- Ignore the "compatible with determinism" target.

## Known Weak Replies

- "Agent causation is mysterious."
- "Everything has a cause, so agent causation is false."
