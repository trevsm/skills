# Commitments

- The suite tests free will as desert-responsibility, not voluntary action.
- Frankfurt cases challenge alternative possibilities but not ultimate sourcehood.
- Randomness does not create desert-responsibility.
- Agent causation is outside deterministic causal closure and therefore does not rescue compatibility with determinism.
- Useful responsibility practices can survive even if desert-responsibility fails.
- Compatibilist freedom may name a useful replacement concept, but replacement is not preservation of the target thesis.
